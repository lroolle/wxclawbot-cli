** Notes
*** Key Findings
- No .claude-plugin or .agent-plugin format exists — SKILL.md IS the manifest
- ClawHub bundles are zip archives with sha256 fingerprinting uploaded to Convex backend
- $CLAWHUB_WORKDIR env var controls install extraction target (default: ./skills/)
- clawhub inspect runs llmAnalysis + staticScan + vtAnalysis before install

*** Gotchas
- clawhub-publisher has package.json referencing index.js not present in the published bundle — verify all referenced files exist in bundle before publishing
- clawhub-search-verify claims "no-exec" in SKILL.md but cli-wrapper.sh executes clawhub binary — security verdict is post-install, not pre
- ClawHub llmAnalysis verdict is advisory only; Suspicious does not block install
